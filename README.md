# Global Airline Disruption Dashboard -2026 Iran-US War
## project_Overview
This project analyze airline disruptions during the Iran-Us conflict. An interactive dashboard was created in Microsoft Excel to visualize key metrics such as daily airline losses, cancelled flights, passengers impact, and rerouted flights
## Objectives
- Analyze airline disruption data
- Identify top airlines with highest losses
- Find countries most affected by flight disruption
- Understand passengers impact during the conflict
## Tools_Used
- Microsoft Excel
- Pivot Tables
-Charts(Bar,Pie,Line,Combo)
- Slicers for filtering
 ## Dashboard_Insights
 -Identify top Airlines with highest daily losses
- Highlighted countries most affected by flights disruptions
- Visualization passengers impact and cancelled flights
